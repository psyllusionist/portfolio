SmartBrain

Facial Recognition app using React framework and Node.js at the back-end. The most interesting challenge for me as a front-end developer was to create the server that will handle any requests from the app, including user registration, user data change, or image processing. To detect faces in the images, I used the Face Detection Model provided by the Clarifai API. This service uses artificial intelligence to recognize visual content. The user submits an image, and the coordinate locations of where the faces might appear are returned. In this project, I learned Express.js and Knex.js to handle server data. A also learned about storing passwords securely using encryption.

MovieBuddy

The objective of this project was to build an app that allows users to find movies they like. This app is built in React and uses Node.js with Express.js at the back-end. I used OMDb API for searching movies. Since OMDb API returns pages, I also added pagination when there are too many results. Users can register and save their favorite movies to the watchlist. I used Knex.js to handle server data.
